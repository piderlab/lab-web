# 実証的ソフトウェア工学研究室

[信州大学 工学部 実証的ソフトウェア工学研究室](https://www.shinshu-u.ac.jp/faculty/engineering/eict/laboratory/cat/hata_hideaki/)の公式Webサイトです。

本研究室では、ソフトウェア開発プロジェクトを持続可能なものとすることを目指し、実証的手法（エンピリカル・アプローチ）を用いてより良いソフトウェア開発方法を探究しながら、ツールやシステムの開発を進めています。特にソフトウェア資産や人材など、多様な要素を含むソフトウェア・エコシステムに関する大規模データを活用した、新しいソフトウェア保守技術の開発に取り組んでいます。また、近年の生成AI技術の発展に伴う新しいソフトウェア開発と保守のアプローチについても検討しています。これらに限定せず、学生の興味に応じて、現代的なソフトウェア開発に関連するさまざまなテーマの研究に取り組んでいます。

エコシステム内での人々の協力を促す仕組み作りの一環として、[乗り捨て可能カーシェアリング](https://naist-carshare.github.io/)の研究も実施しました。

国際共同研究で実施している、[ソフトウェア業界でのキャリア変遷に関するコホート研究（長期間の追跡調査）への協力](/survey/)をお願いしています。
