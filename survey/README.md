# ソフトウェア業界でのキャリア変遷に関するコホート研究（長期間の追跡調査）への協力のお願い

信州大学の畑秀明と申します。 ソフトウェア業界でのキャリア変遷に関するコホート研究（長期間の追跡調査）の協力をお願いしております。

本調査はソフトウェア業界での仕事に対する認識とそれが時間とともにどのように変化するかを調査することを目的とした、複数年に及ぶ長期調査です。カナダ・ダルハウジー大学の Miikka Kuutila 博士が主導し、複数言語で用意したオンライン調査で実施しております。私が日本語版を担当しております。

プロジェクトのWebページはこちらです。<br>
https://paulralph.name/career_progression/

対象は18-65歳でソフトウェア開発業務に携わっている方です。調査にご協力いただける方は次のページからお願いします。<br>
https://surveys.dal.ca/opinio/s?s=80262&lang=ja
上記Webページより他の言語を選んでいただいても大丈夫です。

大学卒業・大学院修了前の事前登録にご協力いただける場合はこちらからお願いします。<br>
https://surveys.dal.ca/opinio/s?s=80550&lang=ja

ご検討のうえ、よろしければご参加をお願いいたします。また、周りの方々にもお知らせいただけますと幸いです。 

ご参考までに、以前同様の研究グループで「COVID-19期間中の在宅勤務がソフトウェア開発に携わる専門家にどのような影響を与えているかの調査」についてのオンライン調査をさせていただいた結果は以下のように発表しております。<br>
https://doi.org/10.1007/s10664-020-09875-y<br>
http://www.naist.jp/pressrelease/2020/05/007025.html

本調査の日本語版についてのお問い合わせは、hata (at) shinshu-u (dot) ac (dot) jp（信州大学 畑秀明）までお願いします。ご参加いただいた場合、お求めがございましたら、研究発表以前であっても可能な範囲で途中経過などを共有させていただきたいと考えております。何卒よろしくお願い申し上げます。
